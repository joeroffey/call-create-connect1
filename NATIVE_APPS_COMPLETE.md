## 🔧 **BUILD & DEPLOYMENT COMMANDS**

### **Development**
```bash
# Web development
npm run dev

# iOS development (requires macOS)
npm run ios:dev

# Android development
npm run android:dev
```

### **Production Builds**
```bash
# Build all platforms
npm run build:prod

# iOS production
npm run ios:build

# Android production  
npm run android:build
```

```bash
# Clean build for stores
npm run mobile:clean

# Sync all platforms
npm run mobile:sync
```
